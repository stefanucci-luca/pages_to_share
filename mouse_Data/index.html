Author

Frontini lab - Data analyst: Luca contact:ls31@sanger.ac.uk

Published

January 9, 2023

There are two batches of mouse data. The first experiments were perormed by Rita Tomè, the second batch of experiments have been performed by Guillaume Bidault (gb483@medschl.cam.ac.uk).

Some lines of the code are coming from https://www.statology.org/modified-z-score/, https://stats.stackexchange.com/questions/242109/model-failed-to-converge-warning-in-lmer and https://cran.r-project.org/web/packages/multilevelTools/vignettes/lmer-vignette.html

Phenotype: mouse weight

The analysis for weight are performed gender stratified. At first looking at the effect using all the mice (i.e Rita’s and Guillaume’s experiments). Batch has been regressed out as covariate when aggregating Rita’s and Guillaume’s data.

There are 3 different data transformation/normalisation for weight measurements:

  1. weight increment over time
  2. Z-score - return the output of the analysis in standard deviation.
  3. median absolute deviation (MAD) Z-score - which is more robust given the number of individuals in the experiments

For each of the transformation, I looked at genotype effect using 2 regression models:

  1. linear model
  2. linear mixed-effect model. This model allow to estimate genotype effect with random intercept and slope, hence giving the possibility to include in the model inter-individual variability (i.e. each person would have a slighlty different response to the genotype).

Intraclass correlation coefficient (ICC) - Female mice only

ICC is a measure of the proportion of variance that is between people versus the total variance (i.e., variance between people and variance within persons). An ICC of 1 indicates that 100% of all variance exists between mice, which would mean that 0% of variance exists within mice, indicating that people have identical scores every time they are assessed. Conversely an ICC of 0 would indicate that everyone’s average was identical and 100% of the variance exists within person.
